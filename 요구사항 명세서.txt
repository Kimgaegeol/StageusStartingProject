플랫폼 : 프로그램
사용 언어 : 파이썬
GUI : pyqt
데이터베이스 : SQLite
GUI 크기 : 1600 x 900
디자인 명세 : 넷플릭스나 유튜브 비슷

1. 로그인 페이지
 <기본 기능 및 이벤트> 
  -  로그인 : 데이터베이스에 저장된 올바른 아이디와 비밀번호 입력 후 로그인 버튼 클릭 시 재생목록으로 넘어감
                틀릴 시 재생목록 페이지로 이동 불가 및 로그인 실패 알림창 띄워짐
  -  회원가입 버튼 : 회원가입 페이지로 넘어감
  -  개인정보 찾기 버튼 : 개인정보 찾기 페이지로 넘어감
  

2. 회원가입 페이지 
 <기본 기능 및 이벤트>           
  - 회원가입 : 개인정보 칸에 개인정보 입력 후 회원가입 버튼 클릭 시 회원가입 성공 
                   (개인정보 :  아이디, 비밀번호, 이름, 전화번호)
  - 뒤로가기 버튼 : 전 페이지로 넘어감
 <예외 처리> 
  - 아이디 중복 확인 버튼 : 아이디 입력 후 클릭 시, 데이터베이스에 저장된 아이디와 비교하고 중복 유무를
                                    알림창을 통해 확인, 중복이 없을 시에만 회원가입 가능
  - 비밀번호 정보 보호 : 비밀번호 입력창에 비밀번호 입력 시 문구 모두 별로 표시 
  - 비밀번호 확인 : 비밀번호 확인 칸 추가하여 입력한 비밀번호와 비교하고 맞는 지 틀린 지 알림창을 통해
                          확인, 맞을 시에만 회원가입 가능 
  - 아이디 예외처리 : 
  - 비밀번호 예외처리 :
  - 이름 예외처리 : 
  - 전화번호 예외처리 : 

3. 개인정보 찾기 페이지 
 <기본 기능 및 이벤트> 
  - 아이디 찾기 : 이름, 전화번호 칸에 개인정보 입력 후, 아이디 찾기 버튼 클릭 시 데이터베이스의 정보와 비교하여 
                       맞는 아이디가 있을 시 알림창을 통해 확인, 만약 없다면 알림창을 통해 없음을 확인 
  - 비밀번호 찾기 : 이름, 전화번호, 아이디 칸에 개인정보 입력 후, 비밀번호 찾기 버튼 클릭 시  데이터베이스의 정보와
                           비교하여 맞는 비밀번호 있을 시 알림창을 통해 확인, 만약 없다면 알림창을 통해 없음을 확인 
  - 뒤로가기 버튼 : 전 페이지로 넘어감 

4. 재생목록 페이지 
 <기본 기능 및 이벤트> 
  - 재생목록 추가 : 회원가입 후 처음 들어오면, 재생목록칸이 비어있으며, 추가 버튼 클릭 시,  재생목록의 이름 설정 칸이 
                         나오며 설정 완료 시 갱신된 재생목록이 뜸
  - 재생목록창 띄움 : 사용자가 접속 시 세로로 2 가로로 n 배열 형태로 재생목록의 첫번째 동영상 썸네일이 뜸
                            ( 재생목록이 한 개라면 1 x 1 배열 형태 )
  - 재생목록 실행 : 재생목록 오른쪽 상단의 재생 버튼 클릭 시 재생목록에 동영상 페이지로 넘어감
  - 재생목록 삭제 : 재생목록 오른쪽 하단의 삭제 버튼 클릭 시, 확인 알림 창이 나오며 알림 창의 확인 버튼 클릭 시 
                          재생목록이 삭제됨. 
  - 검색 기능 : 재생목록 상단에 검색창 존재, 검색창에 올바른 url을 적고 검색창 오른쪽의 검색 버튼 클릭 시 올바른 url
                   이라는 알림창과 함께 저장버튼 뜸, 만약 틀린 url을 작성 시 실패 알림창과 함께 저장버튼이 등장하지 않음 
  - 동영상 저장 : 저장 버튼 클릭 시 재생목록의 리스트가 뜸, 리스트 중 선택 버튼 누를 시 동영상이 알맞은 재생목록에
                       저장되며 갱신된 저장목록 페이지가 새롭게 뜸 
  - 뒤로가기 버튼 : 전 페이지로 넘어감 (아이디와 비밀번호 칸이 지워져 있어야함) 
  
5. 동영상 페이지 
 <기본 기능 및 이벤트>
  - 동영상 재생 : 모든 영상이 오른쪽에 리스트로 존재하며 리스트의 영상 오른쪽에 재생버튼 클릭 시 영상 재생 
                     ( 영상 제목이 30자를 넘길 시, 나머지 부분은 ...으로 표시 )
  - 동영상 일시 정지 버튼 : 동영상의 우측 하단의 버튼 클릭 시, 영상이 플레이 중이면 일시 정지 버튼, 일시 정지 중이면
                                    플레이 버튼으로 바뀜
  - 동영상 중지 버튼 : 동영상의 우측 하단의 3가지 버튼 중 왼쪽에 존재, 클릭 시, 처음 동영상 재생 상태로 돌아간다. 
  - 영상목록 확인 칸 : 동영상의 오른쪽에 해당 재생목록의 나머지 영상들이 일정한 크기로 리스트로 존재, 만약 리스트의
                             칸을 넘어갈 시, 스크롤을 통해 나머지 확인 가능 
  - 다른 영상 재생 : 영상목록 확인 칸의 영상 썸네일이나 제목 클릭 시, 해당 영상이 재생됨. 
  - 동영상 삭제 : 영상목록 칸의 각각의 영상 오른쪽에 삭제 버튼 클릭 시, 삭제 확인 문구가 나오고 확인 버튼 클릭 시, 
                          해당 동영상이 재생목록에서 삭제됨
  - 뒤로가기 버튼 : 전 페이지로 넘어감
